# 🍽️ Visualize AI

Turn restaurant data into profit.

Visualize AI is an AI-powered restaurant analytics platform that converts raw datasets (CSV/Excel) into visualizations and actionable insights. It helps identify top-selling and low-performing menu items, track profit trends, and optimize business decisions.

## 🚀 Features
- 📊 Automatic charts (bar, line, pie)
- 📈 Sales & profit analysis
- 🍔 Top 2 and Bottom 3 menu items
- 📅 Trend tracking (sales over time)
- 💡 Business insights & recommendations
- 📂 CSV & Excel upload support

## 🛠️ Tech Stack
- Backend: Java (Spring Boot, Maven)
- Frontend: React / Vite
- Charts: Java-based visualization libraries

## ⚙️ Run Locally

Backend:
mvn clean install  
mvn spring-boot:run  

Frontend:
cd frontend  
npm install  
npm run dev  

## 🎯 Use Case
Built for restaurants to analyze sales data, improve menu performance, and maximize profit.
